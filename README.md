# 발주서 제작 프로그램

주문 정보를 입력하고 발주서를 생성하는 웹 앱입니다.

## 로컬 실행

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:5173 으로 접속합니다.

## Vercel 배포

1. 프로젝트를 Vercel에 연결한 뒤 배포하면 됩니다.
2. 기존 Vercel 프로젝트에 추가하는 경우: 해당 저장소에 이 폴더를 넣거나, 서브 경로로 배포할 수 있습니다.

## 기능

- 주문 정보 / 고객(수행인사) 정보 입력
- 주문 종류 선택: POP, 스티커, DTF, 티셔츠, 명함, 배너, 고무판, 기타
- 주문 종류별 상세 옵션 (기호, 세로, 수량 등)
- 시안 이미지 첨부
- 측정값 입력 (목 봉제선, 인쇄 사이즈)
- 메모 공간
- 선택 내용을 발주서로 정리하여 출력
